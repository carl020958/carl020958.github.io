### AWS EC2 SSH
* Instance 생성시 발급받는 key(EC2Tutorial) 잘 보관
* 권한 수정 후 ssh로 접속

```bash
# 권한 수정
chmod 0400 EC2Tutorial.pem

# ssh -i [key] [user]@[Public IPv4 address]
ssh -i EC2Tutorial.pem ec2-user@54.180.108.226
```
