### AWS Redshift - Dbeaver 연결
* IAM Role - AmazonRedShiftFullAccess 추가
* Security Group - Inbound Rule에 Type: Redshift, My IP 추가
* Redshift Cluster 생성 후 Actions의 Modify publicly accessible setting에서 Enable로 바꿔주기
