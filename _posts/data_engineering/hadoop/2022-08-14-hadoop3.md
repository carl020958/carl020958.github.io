### Haddop CLI
```bash
# 폴더 생성(-p: 상위 폴더가 없으면 자동으로 생성)
hdfs dfs -mkdir -p /user/ml-100k

# 로컬의 파일을 HDFS에 복사(-f: 같은 이름의 파일 존재할 시 덮어쓰기)
hdfs dfs -put -f u.data /user/ml-100k
```
