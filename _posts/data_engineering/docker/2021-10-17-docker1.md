### Docker 관련 명령어

```docker
# create container(docker container run -d(background) -p[localhost port:container port] [image_name] --name [name_of container])
docker container run -d -p 80:80 nginx --name webserver

# list of running containers (give option -a to sell all containers)
docker container ls -a

# show logs of container(docker container logs [name_of_container/ container id])
docker container logs webserver1

# show process list of container(docker container top [name_of_container/ container id])
docker container top webserver1

# stop container (docker stop [name_of_container/ container_id])
docker stop webserver1

# remove container (docker rm [name_of_container/ container_id])
docker rm webserver1
```
