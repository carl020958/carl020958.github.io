### SQL 주의사항
* COUNT(변수)는 null을 count하지 않음
* COUNT(숫자)는 숫자에 상관 없이 전체 행의 개수 반환

```sql
%%sql
SELECT COUNT(1), COUNT(value), COUNT(DISTINCT value)
FROM testTbl;
```

%%sql

FROM testTbl
WHERE value IS NULL;
