### 문제
* [🔗 문제 링크](https://www.acmicpc.net/problem/10829)

### 코드
```python
def binary(n):
  # 2보다 작으면 재귀함수 종료
  if n < 2:
      return str(n)
  # 그렇지 않다면 n을 2로 나눈 수(나머지 없이)에 대해 재귀함수를 다시 호출 + n을 2로 나눴을 때 나머지를 더함
  else:
      return binary(n // 2) + str(n % 2)

if __name__ == "__main__":
    n = int(input())
    print(binary(n))
```
