### 문제
* [🔗 문제 링크](https://www.acmicpc.net/problem/10162)

### 코드
```python
def solution(T):
    if T % 10 != 0:
        return [-1]
    
    A, B, C = 300, 60, 10
    a, b, c = 0, 0, 0    
    while T:
        if T >= A:
            a += 1
            T -= A
        elif T >= B:
            b += 1
            T -= B
        elif T >= C:
            c += 1
            T -= C
        
    return [a, b, c]

if __name__ == "__main__":
    T = int(input())
    print(*solution(T))
```
